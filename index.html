Who are we?

Indiana State Police Roleplay (or INSPRP) is a serious FiveM RP Server, with many unique opportunites and a great community to back that up!

With an active staff team and an outgoing, yet friendly community, we hope this will be your server of choice.

Where Do I Start?

Firstly, make sure the server is running at top notch speed by checking out our Status Page.


Secondly, make sure you check out the New Players Information, as it contains everything, from how to be able to join our server to how our commands work
